File Directory Breakdown

Frontend (Angular)

  • speech.service.ts: Manages the browser's Web Speech API.
  • app.ts: Coordinates recording, saving, and loading data.
  • app.html/css: The user interface and visual styling.

Backend (NestJS)

  • app.controller.ts: Handles incoming API requests (POST/GET).
  • app.service.ts: Contains the logic for reading/writing CSV files.
  • main.ts: Configures the server (CORS, Port 3000).

Python & Data

  • scripts/clean_data.py: Cleans raw CSV data (lowercase, trim).
  • data.csv: The "living" file where raw text is appended.
  • cleaned_data.csv: The "normalized" version for analysis.

Quick Start Guide

  1. Start Backend: cd backend && npm run start:dev
  2. Start Frontend: npm start
  3. Speak in the app and click Save.
  4. Run python scripts/clean_data.py in your terminal.
  5. Click Load Cleaned Data in the app to see the magic!
